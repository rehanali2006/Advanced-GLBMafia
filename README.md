To run this project-
1) You have to first copy the project in your computer.
2) Install dependencies using npm install
3) start redis server in wsl by typing redis-server
4) now start the server node app.js
5) got to localhost:8000


The environmant variables are in this format

MONGO_URL=
SESSION_SECRET=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLOUDINARY_FOLDER=
